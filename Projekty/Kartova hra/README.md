# Kartová hra

Bojová hra vytvorená v Pythone s pygame-ce.

## Spustenie

1. **Nainštaluj pygame-ce**

   pip install pygame-ce

2. **Spusti hru**

   python main.py

## Ako hrať

1. Vyber si level
2. Zostav si armádu za dostupné peniaze
3. Bojuj proti nepriateľom!

### Jednotky

| Jednotka | HP  | Útok | Špeciálne          | Cena |

| Warrior  | 50  | 5    | -                  | 20   |
| Knight   | 50  | 7    | -                  | 30   |
| Defender | 60  | 3    | Blokuje 2 DMG      | 35   |
| Vampire  | 40  | 4    | Lieči sa 50% z DMG | 40   |
